
  ## Running the code

  Run `npm i` to install the dependencies.

  Run `npm run dev` to start the development server.
  
## Contact form email setup (Vercel + Resend)

Set these environment variables in Vercel Project Settings:

- `RESEND_API_KEY`: Your Resend API key.
- `CONTACT_FROM_EMAIL`: Verified sender address in Resend (example: `Portfolio Contact <onboarding@resend.dev>` during testing).
- `CONTACT_TO_EMAIL`: Recipient mailbox (the client Gmail address).
